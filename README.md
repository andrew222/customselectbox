customselectbox
===============

A jQuery plugin to custom your select box

### How to use
##### html
	<select id="item">
	  <option>option1</option>
	  <option>option2</option>
	</select>
##### Javascript
	$("#item").customselectbox({position: "after"});
##### css
	.customselectbox {
		padding: 10px;
	}
